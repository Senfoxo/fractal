Key Features

  • Support for functional programming paradigms
  • Efficient memory management
  • Seamless integration with existing libraries

Function Reference

  • <^> : Import a module
  • <#> : Define a constant value
  • <$> : Define a function for printing
  • <&> : Define the main program
  • <%> : Call the main program
  • <!> : Define a factorial function
  • <@> : Define a recursive function
  • <~> : Define a higher-order function
  • <*> : Define a generator function
  • <+> : Define an asynchronous function
  • <_> : Define a function with default parameters
  • <%!> : Define a memoized function
  • <$@> : Define a curried function
  • <!$> : Define a function with rest parameters
  • <@!> : Define a function with pattern matching
  • <%*> : Define a function with spread operator
  • <+@> : Define a function with async/await
  • <_!> : Define a function with optional chaining
